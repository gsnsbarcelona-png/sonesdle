Adivina el Pro Player

Un jugador misterioso se esconde cada día. Adivínalo usando las pistas que te da cada intento: país, posición, equipo, edad, liga y más.

  • Celda verde = coincide exactamente
  • Naranja = parcialmente correcto
  • Rojo = no coincide
  • Flechas indican si el valor es mayor o menor
Jugar 02 🔲
Grid · Conexiones

LoL Pro Grid

Una cuadrícula 3×3 con criterios en filas y columnas. Encuentra el pro player que cumple ambas condiciones de cada casilla. Tienes vidas limitadas.

  • 9 casillas para rellenar
  • Combina equipo, posición, liga, país…
  • Vidas limitadas por partida
  • Sin repetir jugadores
Jugar 03 📋
Timeline · Carrera

LoL Career Guess

Se muestra la carrera de un pro player con equipos y años. Adivina quién es a partir de su historial profesional paso a paso.

  • Historial de equipos revelado progresivamente
  • Cuantos menos pasos, mejor puntuación
  • Datos de toda la carrera del jugador
  • Jugadores de todas las ligas principales
